## Clone this project

```bash
> git clone https://github.com/Ivan-MLN/SELF-BOT
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> pkg install unzip 
> unzip SELF-BOT.zip
> cd SELF-BOT
> bash install.sh
```

### Usage
```bash
> npm start 
```
